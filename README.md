# e-assignment

Exec `make run-compose` or `docker-compose up --build` to start server

.env file configures server with env variables

E_LOG_LEVEL (0 - 6) - set log level

E_LOG_FILE - put logs to file (std dy default)

E_DB_DSN - dsn for database

E_PORT - port of app

API doc available on http://localhost:8080/swagger/index.html by default
